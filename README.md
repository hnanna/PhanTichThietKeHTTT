# Phân Tích & Thiết Kế Hệ Thống Thông Tin

## Mô tả dự án
Dự án tập trung vào phân tích nghiệp vụ, thiết kế hệ thống và xây dựng ứng dụng quản lý nhân viên bằng Windows Forms dựa trên các tài liệu phân tích đã thực hiện.

## Nội dung 
- Use Case Diagram 
- Workflow / Activity Diagrams  
- Entity Relationship Diagram (ERD)  
- Sơ đồ luồng màn hình (Screen Flow)  
- Ứng dụng Quản lý nhân viên xây dựng bằng Windows Forms
- Tài liệu và báo cáo dự án

## Công nghệ & công cụ sử dụng

- SQL Server
